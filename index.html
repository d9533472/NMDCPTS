📋 Setup Guide

Step 1: Create a new Google Spreadsheet
Step 2: Go to Extensions → Apps Script
Step 3: Paste the Apps Script code (apps_script_backend.js)
Step 4: Run initSheets() once to create sheet headers
Step 5: Deploy → New deployment → Web App
    • Execute as: Me
    • Who has access: Anyone
Step 6: Copy the Web App URL and paste above
Step 7: Click Push Local → Cloud to upload existing data
